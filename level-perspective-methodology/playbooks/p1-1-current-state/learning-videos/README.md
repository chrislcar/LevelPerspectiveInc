# p1-1-current-state/learning-videos/

LEARNING video scripts. ≤5 minutes each. Delivered async per the flipped-classroom commitment; sync sessions contain zero teaching minutes.

Each video has three sub-artifacts:
- **Script** (here, as `.md`)
- **Slide visuals** (in Loop or Figma; URL captured in `/visual-masters/`)
- **Recorded video** (in Loop / SharePoint; URL captured in `/visual-masters/`)

## Curriculum

| File | Video | Length | Type | Step it primes |
|---|---|---|---|---|
| `v1-script.md` | Welcome & Journey 1 orientation | 3 min | Engagement framing | Step 1 |

## Script structure

Each script:
- Opens with the failure mode the video prevents
- Names the framework or anti-pattern in canonical Strategyzer/H&S/Schein language
- Gives one worked example (PathLine where possible for cross-step continuity)
- Closes with the named output the viewer should be able to recognize/produce

Length discipline: assume ~150 spoken words per minute. A 5-min video is ≤750 words of script.

## Production order

1. Script first (here)
2. Slide visuals second (Loop or Figma)
3. Recording last (Loop)

This order minimizes rework. Scripts that change after recording cost an order of magnitude more time than scripts that change before.
