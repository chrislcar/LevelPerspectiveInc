## Use during engagement

When an engagement spawns (per the spawn-engagement ritual in `/canonical-prompts/daily-ops/`), run-kit artifacts get copied into the engagement workspace and customized per client. The originals here are evergreen templates.

## Versioning

Run-kit artifacts evolve based on engagement learnings. Per-engagement retros (run by canonical engagement-close prompt) flag run-kit updates as methodology backlog items. Updates are versioned through the standard semver bump rules.
