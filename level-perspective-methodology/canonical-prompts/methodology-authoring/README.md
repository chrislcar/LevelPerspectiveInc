## When to use

These prompts are for the methodology-author seat (you, in this Claude Project, building the offering). Not for engagement delivery.

## Pattern

Each prompt enforces:
- Inheritance of canonical patterns where they exist (don't redesign what's already settled)
- Explicit divergence rationale where novel structure is required
- Methodologist surfaces / leadership owns / AI structures discipline
- Sticky color / evidence ladder / postcard altitude vocabulary
- Citation of source (project knowledge, Open Brain, inference, hypothesis)
